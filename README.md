## Obodom Integrated Farms

Landing site for Obodom Integrated Farms — showcasing regenerative agriculture, export-ready produce, and community-first programs.

### Stack
- Next.js 16 (App Router), React 19, TypeScript
- Tailwind CSS 4 with framer-motion for animations

### Run locally
```bash
npm install
npm run dev
visit http://obodomfarms.com
```

### Scripts
- `npm run dev` – start the dev server
- `npm run build` – production build
- `npm run start` – run the built app
- `npm run lint` – lint the project
